Komodo xJSFL Extension

08Jul2011 0 Comments

Overview

Komodo Edit is extremely customisable, and the Komodo xJSFL extension adds functionality that makes it easier than ever before to develop JSFL.

Its features include:

  • Full autocomplete for the JSFL API
  • A variety of keyboard shortcuts to run and execute JSFL
  • Integrated debugging capabilities
  • Code abbreviations for JSFL and XUL
  • XUL panel previewing

As a developer, using Komodo plus the extension enables you to write significantly better, faster, more organised JSFL than ever before.
